# Flores Amarillas Mágicas 🌻✨

¡Bienvenido al código fuente de este pequeño detalle especial!

Este proyecto es una animación web romántica construida con **React** y **Vite**, diseñada para sorprender a esa persona especial. Al abrir la aplicación, el usuario es recibido por una interfaz cristalina (Glassmorphism), un botón palpitante que, al presionarlo, despliega un campo de 16 flores amarillas floreciendo sincronizadamente bajo un cielo nocturno animado con lluvia de estrellas y luciérnagas.

## ✨ Características Principales
- **Animaciones CSS Complejas:** Crecimiento escalonado de las flores, pasto, pétalos y destellos mágicos.
- **Interactividad:** Efectos hover (resplandor) en las flores al pasar el cursor.
- **Efecto Máquina de Escribir:** Un hermoso mensaje que se revela carácter a carácter con pausas y saltos de línea dramáticos.
- **Música de Fondo:** Acompañamiento a elección.

## 🚀 Cómo ejecutar el proyecto (Modo Desarrollador)

1. Asegúrate de tener Node.js instalado.
2. Clona este repositorio o descarga el código.
3. Instala las dependencias ejecutando:
   \`\`\`bash
   npm install
   \`\`\`
4. Levanta el servidor local con:
   \`\`\`bash
   npm run dev
   \`\`\`
5. ¡Abre la dirección \`http://localhost:5173/\` en tu navegador y disfruta de la magia!
